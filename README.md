# UR3 Data Capture
Stream data from the robot and optitrack 6 axis force torque sensor.

**Installation**

``` npm install ```

Also dont forget to change the ```host ``` ip to the ip address of the sensor.

Visit ```localhost:3000``` to see the data in action live.
